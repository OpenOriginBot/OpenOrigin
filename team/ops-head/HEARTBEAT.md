# HEARTBEAT.md - 运营部心跳巡检

## 每小时核查

- [ ] 是否有新支付订单需处理
- [ ] 是否有订单状态变更（已发货/已完成）
- [ ] 是否有紧急物流查询请求

---

## 每日核查（固定时间：09:00、14:00、20:00）

### 发货管理
- [ ] 统计当日待发货订单数量
- [ ] 核查48h内未发货订单
- [ ] 复核发货延迟订单原因
- [ ] 确认当日发货完成率

### 库存管理
- [ ] 核查库存预警SKU（低于3天销量）
- [ ] 确认缺货SKU是否有在途订单
- [ ] 复核新增待入库数量
- [ ] 整理当日库存变动台账

### 物流管理
- [ ] 统计当日发货物流单号
- [ ] 核查物流轨迹异常订单
- [ ] 复核买家物流咨询反馈
- [ ] 确认签收异常处理进度

---

## 每周核查（每周一）

### 发货表现
- [ ] 统计上周48h发货率
- [ ] 分析发货延迟原因分类
- [ ] 评估专线/爆品发货表现
- [ ] 制定本周发货提升计划

### 库存健康
- [ ] 统计上周库存周转天数
- [ ] 识别滞销SKU（30天无动销）
- [ ] 复核安全库存设置合理性
- [ ] 制定本周补货计划

### 供应商表现
- [ ] 统计各供应商到货准时率
- [ ] 记录供应商交期异常
- [ ] 同步给营收部供应商绩效
- [ ] 跟进待处理供应商问题

### 物流表现
- [ ] 统计上周物流异常率
- [ ] 分析物流投诉原因
- [ ] 评估物流服务商表现
- [ ] 制定物流改善计划

---

## 每月核查（每月1日）

### 综合评估
- [ ] 汇总上月运营KPI达成情况
- [ ] 分析重大异常事件
- [ ] 评估运营成本效率
- [ ] 制定本月运营重点

### 系统维护
- [ ] 备份上月运营数据
- [ ] 清理过期临时文件
- [ ] 更新运营SOP文档
- [ ] 复盘上月决策记录

---

## 预警阈值

| 指标 | 红色预警 | 黄色预警 | 处理措施 |
|------|----------|----------|----------|
| 48h发货率 | <90% | <93% | 立即排查+上报CEO |
| 库存预警SKU | ≥10个 | ≥5个 | 立即启动补货+通知营收部 |
| 物流异常单 | ≥5单/天 | ≥3单/天 | 核查物流商+上报 |
| 断货SKU | ≥3个 | ≥1个 | 立即通知CEO+营收部 |
