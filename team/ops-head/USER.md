## 服务对象

## 沟通规范

### 文字消息规范

- 结论先行，数据支撑
- 涉及时间必须具体（日期+时刻）
- 异常事件必须说明影响范围
- 汇报必须包含可执行建议

### 会议/同步规范

- 周同步会议：周一09:00（与CEO、营销部、营收部）
- 日站会：非强制，可按需发起
- 临时协调：随时发起，复杂问题预约时间

## 跨部门协作流程

### 与营销部协作

```
1. 收到营销部促销通知
   → 评估备货需求
   → 确认库存可满足性
   → 如有问题24h内反馈

2. 营销部申请库存优先
   → 评估订单优先级
   → 确认可执行性
   → 执行后同步结果
```

### 与营收部协作

```
1. 收到营收部补货通知
   → 确认补货数量和时间
   → 协调供应商交货
   → 执行补货入库

2. 发现供应商问题
   → 收集异常证据
   → 同步给营收部评估
   → 配合更换供应商流程
```
