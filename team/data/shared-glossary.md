# 共享术语表

## 业务术语

| 术语 | 定义 | 适用部门 |
|------|------|----------|
| 48h发货率 | 订单付款后48小时内发货的比例 | 运营部 |
| SKU | 库存量单位，本项目中指灯带具体规格型号 | 全体 |
| DOA | 到货即损（Dead On Arrival），收货时已损坏 | 运营部/营销部 |
| GMV | 商品交易总额 | 营收部 |
| 毛利率 | （售价-成本）/售价 × 100% | 营收部 |
| 转化率 | 访客到下单的比例 | 营销部 |
| 客单价 | 每笔订单的平均金额 | 营收部 |
| 退款率 | 退款订单占总订单的比例 | 全体 |

## 灯带专业术语

| 术语 | 定义 |
|------|------|
| 色温 | 光的颜色，暖白光(3000K)/正白(4000K)/冷白(6000K) |
| LED密度 | 每米灯珠数量，常见30灯/60灯/120灯 |
| IP等级 | 防水等级，IP20(室内)/IP65(防泼溅)/IP67(防水) |
| RGB | 红绿蓝三色，可变色 |
| 单色 | 单一颜色灯带 |
| 剪裁单元 | 灯带可剪裁的最小长度单位 |

## 平台规则

| 规则 | 说明 |
|------|------|
| 48小时发货 | 1688要求订单付款后48小时内发货 |
| 7天无理由 | 1688买家享有的退货权利 |
| 品质退款 | 因质量问题产生的退款，不计入商家违规 |
| 好评率 | 店铺评分的重要指标，目标≥98% |

## 紧急程度定义

| 等级 | 响应时间 | 示例 |
|------|----------|------|
| P0-紧急 | 立即 | 断货危机、差评风暴、资金异常 |
| P1-高优 | 30分钟内 | 发货延迟、库存预警、客户投诉 |
| P2-常规 | 2小时内 | 日常询价、内容更新、数据整理 |
| P3-计划 | 24小时内 | 周报撰写、活动策划、竞品分析 |
