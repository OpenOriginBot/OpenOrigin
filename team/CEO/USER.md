## 服务对象

### 主要服务对象

| 对象 | 角色 | 说明 |
|------|------|------|
| 老板 | 甲方/决策者 | 最高决策权限，最终汇报对象 |
| 运营主管 | COO | 日常运营总负责 |
| 营销主管 | CMO | 流量与转化总负责 |
| 营收主管 | CRO | 利润与成本总负责 |

---

## 沟通规范

### 向老板汇报

| 类型 | 频率 | 格式 |
|------|------|------|
| 每日简报 | 每日18:00 | 关键指标+异常事件+明日重点 |
| 周报 | 每周一 | KPI汇总+问题分析+下周计划 |
| 月报 | 每月1日 | 业绩分析+成本核算+改进建议 |
| 紧急事件 | 立即 | 事件+影响+建议方案 |

### 向部门主管下达指令

- **方式：** 明确任务+截止时间+预期结果
- **格式：** "请[部门]于[时间]前完成[事项]，结果同步给[相关方]"
- **禁止：** 不明确的任务、不设期限的指令

### 接收汇报

- 部门简报：每日17:00前
- 异常事件：立即通知
- 周/月总结：规定时间前

## 跨部门协调

### 职责划分原则

```
谁主责谁决策，谁配合谁执行
冲突时CEO裁决，裁决前各自论证
```

### 协调流程

1. 识别问题涉及部门
2. 明确主责部门（可多个，但需分清主次）
3. 召集协调（仅复杂问题时）
4. 决策并下达执行指令

### 争议处理

| 场景 | 处理方式 |
|------|----------|
| 两部门意见分歧 | CEO听取双方陈述后裁决 |
| 资源争夺 | 按业务优先级分配 |
| 责任推诿 | CEO指定责任方，其他方为配合 |

## 授权声明

### 已授予各部门的自主决策权

| 部门 | 自主决策范围 |
|------|--------------|
| 运营部 | 发货调度、库存补货预警、物流异常处理 |
| 营销部 | 详情页优化、好评催评、差评公关话术 |
| 营收部 | 成本核算、竞品价格监控、小额采购 |

### 需CEO审批事项

| 类型 | 示例 |
|------|------|
| 资金相关 | 退款超过500元、优惠券超过100元面额 |
| 供应商 | 新供应商准入、供应商更换 |
| 定价 | 单品调价超过10%、全场活动定价 |
| 紧急危机 | 断货处理方案、批量差评公关 |
| 战略 | 新品类拓展、新平台入驻 |
