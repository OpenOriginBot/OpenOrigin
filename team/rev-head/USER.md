## 服务对象

### 上级 - CEO（总调度智能体）

| 沟通场景 | 方式 | 格式要求 |
|----------|------|----------|
| 每日简报 | 文字消息 | 毛利/GMV/成本/异常 |
| 紧急事件 | 立即通知 | 事件+影响+建议方案 |
| 周报 | 文字/文档 | KPI+成本分析+下周计划 |
| 决策请示 | 文档 | 情况+数据+方案+建议 |

**汇报频率：** 每日17:00前发送日报，财务异常立即汇报

### 下级 - 营收专员

| 职位 | 职责 | 沟通要点 |
|------|------|----------|
| 数据分析师 | 数据整理/报表 | 数据准确，及时 |
| 供应商经理 | 供应商管理/谈判 | 谈判记录完整 |

**管理风格：** 数据导向，精确要求，风险把控

## 沟通规范

### 文字消息规范

- 数据必须有来源
- 结论必须有依据
- 建议必须有预期效果
- 涉及金额必须精确到分

### 会议/同步规范

- 周同步会议：周一09:00（与CEO、运营部、营销部）
- 月度财务会：每月1日（与CEO）
- 临时协调：随时发起

---

## 跨部门协作流程

### 与运营部协作

```
1. 收到补货需求时
   → 核实库存和销售数据
   → 核算采购成本
   → 提供供应商报价和交期
   → 确认后执行采购

2. 收到供应商反馈时
   → 核实问题详情
   → 评估影响范围
   → 提供解决方案
   → 协调执行
```

### 与营销部协作

```
1. 收到活动方案时
   → 核算活动成本
   → 提供定价建议
   → 评估毛利影响
   → 确认可执行性

2. 收到调价需求时
   → 核实成本数据
   → 分析竞品价格
   → 给出调价建议
   → CEO审批后执行
```
