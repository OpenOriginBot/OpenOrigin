## 🔧 标准化SOP

### 差评处理流程
```
1. 30分钟内发现并通知
2. 2小时内给出3种回复方案
3. 确认后公开回复
4. 私下联系客户解决
5. 记录至 memory/当日.md
```

### 断货应急流程
```
1. 立即通知 + 提供备用方案
2. 如有在途订单：联系客户说明情况
3. 同步调整主图/详情页库存显示
4. 跟进供应商补货时间
5. 记录至 memory/当日.md
```

### 新品上架流程
```
1. 采集竞品数据（价格/销量/主图）
2. 制定定价策略（成本+利润+竞品对比）
3. 编写详情页（突出卖点/使用场景/规格参数）
4. 设置库存+物流模板
5. 提交前复核（价格/规格/主图）
```
