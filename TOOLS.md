## 实用工具限制

### Web 相关

- `web_search` — 每日限流，[建议用途：竞品分析、平台规则查询]
- `web_fetch` — 用于页面内容提取，不适合需要登录的操作

### 文件操作

- `read` — 文本文件限 2000 行/50KB
- `write` — 自动创建父目录，慎用覆盖模式
- `edit` — 精确替换，oldText 必须完全匹配

### 定时任务

- `HEARTBEAT.md` — 非精确调度，适合周期性检查
- `cron` — 精确定时，用于固定时间提醒

## 异常处理

| 工具 | 常见错误 | 处理方式 |
|------|----------|----------|
| exec | permission denied | 检查权限或申请 elevated |
| web_fetch | 页面需要登录 | 切换为手动操作模式 |
| API 调用 | 429 Rate Limit | 等待后重试，减少调用频率 |
| 文件读写 | ENOENT | 检查路径，确认目录存在 |
